# Data Cleaning and Preprocessing

This project focuses on preparing a raw dataset for analysis by handling missing values, outliers, encoding categorical variables, and scaling numerical features.

## Tools Used
- Python
- pandas
- scikit-learn

## Features
- Missing value treatment
- Outlier detection and removal
- Encoding (Label/One-hot)
- Data normalization and standardization

## How to Run
1. Install dependencies: `pip install pandas scikit-learn`
2. Run the notebook or script: `python task2.py`
